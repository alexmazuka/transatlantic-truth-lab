## Deploy to GitHub Pages (5 minutes)

1. Create a new repository, e.g. `transatlantic-truth-lab`, and upload these files to the **root**.
2. Go to **Settings → Pages**.
3. Under **Build and deployment → Source**, choose **Deploy from a branch**.
4. Select branch **`main`** and folder **`/ (root)`**, then **Save**.
5. Wait ~1 minute; your site goes live at `https://<your-username>.github.io/transatlantic-truth-lab/`.
6. Paste that URL into the grant cover email and proposal.

> Tip: replace the `[email]` placeholders and complete the Team bio in `index.html` before publishing.
